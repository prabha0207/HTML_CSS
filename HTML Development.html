    Html Forms GET vs POST

  • HTTP protocol facilitates communication between client and server
  • two type of data transmission method: post & get

    • post vs get

  • GET request data fro a specified resource
  • POST submits data to be processed to a specified resources

    • GET

  • name/value pairs are send in the URL string
    Eg: Send.php?fname=john&lname=smith

    • POST

  • send name/value pairs in the http message body
  • Eg: POST send.php HTTP/1.1
    fname=john$lnem=smith

    • Get Request

  • Canbe cached
  • Remaining in browser history
  • Should only be used to retrive data
  • Have a length Restrictions
  • Should not be used for sensitive data
  • can be bookmarked

    • Post Request

  • No restriction on data length
  • canot be bookmarked
  • Do not Remain in browser histroy
  • Request are naver cached
